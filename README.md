# vuecli3-ssr-example (server data fetching)

基于VueCLI3 SSR 服务端渲染 

已搭建express node服务器 通过axios进行服务端数据预取渲染首屏 

开箱即用的模板（已添加vue-lazyload）

## Example

安装依赖

npm install

开发 热更新

npm run dev

打包服务器渲染使用

npm run build:win

启动Node

node server
